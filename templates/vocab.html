📥 How to import into Anki

  1. Click Export to Anki above to download the .tsv file
  2. Open Anki → File → Import
  3. Select the downloaded fce_vocab_anki.tsv file
  4. Set Field separator = Tab, Allow HTML = checked
  5. Map columns: Field 1 = Front, Field 2 = Back, Field 3 = Tags
  6. Click Import — done!

Front side = English word + 🔊 pronunciation + sentence, Back side = Russian translation. Tags are auto-set by part number.

Export + Audio downloads a .zip: unzip into your Anki collection.media folder, then import the .tsv file.
